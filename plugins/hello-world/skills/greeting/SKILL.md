# Greeting

This is a minimal example skill shipped by the `hello-world` plugin in the `kccplugin` marketplace.

When invoked:

1. Greet the user warmly.
2. Mention that the greeting is coming from the `hello-world` skill loaded via the `kccplugin` marketplace.
3. Encourage the user to start authoring their own plugins under `plugins/` in this marketplace.
